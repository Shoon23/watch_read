
# KAMM
## Getting Started

First, run the development server:

```bash

npm install
# or
yarn add

#and

npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.




## Tech Stack
    Typescript
    Nextjs
    Tailwind CSS
    Daisy UI
    Mangadex API
    Consumet API
